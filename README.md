# ClimbIt
## 1. Descripción del proyecto
ClimbIt es una aplicación web/móvil que permite a un usuario registrarse para crear su perfil de “escalador” con el cual podrá asignarse a un rocódromo existente. Estos rocódromos añadidos tendrán un registro actualizado de las pistas que hay activas. Estas pistas pueden tener una dificultad asignada de base y además si un escalador marca como completada una pista puede dar su opinión sobre el nivel de dificultad de esta. Completando pistas subirás el nivel de tu perfil y se actualizará estadísticas sobre tu progreso en la escalada permitiéndote compararlo con tu grupo de amigos.
### 1.1. Objetivos
- Permitir a los usuarios registrar detalles de cada sesión
- Proporcionar un perfil personal con historial de progresos
- Conseguir una aplicación intuitiva y simple para que cualquier usuario pueda usarla
- Enganche al usuario final para que quiera mantener su progreso actualizado
- Permitir que los usuarios puedan comparar su progreso con sus compañeros para fomentar una competitividad sana
### 2.2. Metas
- Desarrollar una aplicación móvil funcional para dispositivos iOS y Android.
- Desarrollar una aplicación web accesible desde navegadores modernos.
- Implementar un sistema de gamificación que incluya puntos, niveles y logros.
- Proporcionar estadísticas detalladas sobre el progreso de los usuarios.
